post /authenticate
User Authentication with JWT provided (authenticatePost)
Endpoint used for authorization

Query parameters

email (required)
Query Parameter — Email of app user
password (required)
Query Parameter — Password of app user

Return type

Token

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • text/html
  • application/json

Responses

200

JWT token provided

401

Bad password and email combination
